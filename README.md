# 🌱 Crop Recommendation System Using Supervised Machine Learning

## 📌 Overview
This project recommends the most suitable crop based on environmental and soil conditions using supervised machine learning. It helps farmers make data-driven decisions for better agricultural productivity.

## ✨ Features
- Predicts the best crop based on input values.
- User-friendly interface.
- Machine learning-based prediction.
- Fast and accurate recommendations.

## 🛠️ Tech Stack
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📂 Dataset
The dataset contains parameters such as:
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

## 🚀 How to Run
1. Clone the repository.
2. Install the required libraries.
3. Open the Jupyter Notebook.
4. Run all cells.
5. Enter the input values to get the recommended crop.
